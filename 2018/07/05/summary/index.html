laravel源码(开篇)

发表于 2018-07-05 | 分类于 laravel

版本说明

  • laravel 5.1
  • php 5.6以上

学习心得
  个人觉得看框架源码至少需要有一定的框架使用经验,说白了就是我们知道代码最终执行结果,但是可能不太清楚一些底层流程,这也是个人想去学习的原因，顺便也补一下PHP基础。
  我的学习方法就是拿public\index.php，也就是项目的入口文件当做突破口。就好像,假如没有任何手册，那想要知道为什么就必须花时间和心思去探索。

   index.php 从头到尾不会超过10行代码,总流程 创建App实例、接收request、获取route、返回response，结束

学习的具体实现
  准备一份laravel 源码，再准备一个自己的项目。最开始我们的项目是这样的
|–app 根目录
|–|– public
|–|–|– index.php 拷贝源码
|–|–|– htaccess 拷贝源码
将web服务器指定到该目录下,本地执行。随着系统报错，故事就开始了…
